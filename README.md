# ZLOGGER

Blog platform project

Spring, AngularJS.

<hr/>

**Start web-server**

Use

    `mvn jetty:run`

or

    `mvn tomcat7:run`

<hr/>

**Run all tests**

    `mvn clean verify`

Build test reports (allure)

    `mvn site`

results in `\target\site\allure-maven-plugin.html`
